# 高知ファイナル 3連単ジェネレーター

高知競馬「一発逆転ファイナルレース」専用の3連単買い目ジェネレーター（単一HTML・オフライン動作）。

- AI確率（Plackett-Luce）× 風水・タロット等のオカルトを3軸スライダーでブレンド
- 天気・馬場の現地反映（Open-Meteoで高知の天気を自動取得 / 雨・道悪で荒れ指数↑）
- 🎲 運試しモード（完全ランダム3連単）
- スマホ最適化（ランキングはカード表示・横スクロールなし）

## 使い方
1. このページ（GitHub Pages）をスマホ／PCのブラウザで開く
2. 出馬表をスクショ → AIアプリで「スクショ＋アプリ内のプロンプト」からJSON化 → 貼り付け
3. 「🔥 買い目を生成」。馬場（良/稍重/重/不良）は出馬表の表記に合わせて選択（自動取得も可）

※ 自分用のエンタメ予想ツールです。馬券は余裕資金で。
